# F5: Dashboard

**Purpose:** Overview + intelligence. Scoped to active profile.

## In scope
- **Skills frequency chart** — top N skills across listings (bar chart).
- **Skills trend over time** — line chart (weeks/months).
- **Match alerts** — jobs ranked by weighted fit score, top matches highlighted.
- **Stats cards** — total scraped, new today/this week, % remote, avg salary, most common contract type.
- Alerts are a dashboard section only (no email/push in v1; not excluded later).
